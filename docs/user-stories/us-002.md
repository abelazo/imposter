# Configure Impostor Count

```# User Story
As a game host
I want to set the number of impostors (1 to participants-1)
So that I can adjust game difficulty
```

# Acceptance Criteria
- Display a counter/selector for number of impostors 
- Minimum 1 impostor 
- Maximum is total participants/2 - 1
- Default to 1 impostor

# Implementation Notes
- Use a numeric input or dropdown for selecting impostor count
- Validate input to ensure it stays within the defined range
- Update state to reflect the selected impostor count
- Basic styling for the input/selector
- No backend integration needed for this user story
